Example using Fake GraphQL API


                    http://fake.graphql.guru/graphql/

Query


                    {
                      Fake {
                        id
                        firstName
                        lastName
                      }
                    }

Response


                    {
                      "data": {
                        "Fake": {
                          "id": "c2c2681b-ed9e-42d1-9c3d-228c8d89630d",
                          "firstName": "Dessie",
                          "lastName": "Cremin"
                        }
                      }
                    }

                    val request = Request.Builder()

                            "{\"query\":\"{Fake{id,firstName,lastName}}\"}"))
                        .build()
